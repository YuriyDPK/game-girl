# Warm Memories

Уютный кинематографичный 2D-нарративный прототип браузерной игры на Phaser 3, TypeScript и Vite.

Проект рассчитан на будущую личную романтическую историю: замените заглушки диалогов, добавьте нарисованные фоны и спрайт-листы, затем расширяйте локации и воспоминания — базовые системы менять не нужно.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173/`.

## Структура

- `src/core` — конфиг Phaser, константы, состояние сюжета через registry.
- `src/scenes` — boot, preloader, нарративная сцена, данные локаций.
- `src/entities` — игрок, зоны взаимодействия, профили персонажей.
- `src/ui` — окно диалога, подсказка взаимодействия, переходы с названием сцены.
- `src/audio` — директор эмбиента и музыки.
- `src/dialogues` — сценарии диалогов, ветвящиеся узлы, заглушки сюжетного плана.
- `src/utils` — хелперы цветов и процедурных текстур.
- `src/assets` — будущие фоны, персонажи, объекты, музыка, эмбиент, SFX.

## Как добавить настоящую историю позже

1. Опишите главы в `src/dialogues/storyOutline.ts`.
2. Замените заглушки диалогов в `src/dialogues/scripts.ts`.
3. Добавьте интерактивные воспоминания в `src/scenes/locationData.ts`.
4. Положите финальные арт и аудио в `src/assets` и перечислите файлы в `src/assets/assetManifest.ts`.

Сейчас арт процедурный и намеренно мягкий, «живописный» — чтобы игра была играбельна до появления финальных ассетов.
