# Asset Layout

Future hand-painted or commissioned assets can be dropped into this folder without changing game logic.

- `backgrounds/` - layered location art, parallax plates, atmospheric overlays.
- `characters/` - portrait art, full body sprites, sprite sheets.
- `objects/` - interactable props and memory items.
- `audio/music/` - calm lo-fi and emotional score.
- `audio/ambience/` - rain, cafe room tone, metro hum, city wind.
- `audio/sfx/` - footsteps, soft UI sounds, object details.
